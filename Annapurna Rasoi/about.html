About Annapurna Rasoi

Project Overview

Annapurna Rasoi is a web-based restaurant management system developed using HTML, CSS, and JavaScript. The system allows customers to browse menus, place orders, and enables administrators to manage inventory and orders efficiently.

Objectives

  • Provide a digital ordering system
  • Reduce manual errors in order handling
  • Manage stock automatically
  • Improve user experience

Features

  • Menu browsing with categories
  • Cart and order placement
  • Inventory management system
  • Admin dashboard
  • Persistent data using localStorage

Technologies Used

  • HTML (Structure)
  • CSS (Design)
  • JavaScript (Logic)

Future Scope

This system can be extended with backend technologies like Node.js or PHP and databases like MongoDB or MySQL to support real-time multi-user environments.
