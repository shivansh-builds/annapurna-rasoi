🍽️ Annapurna Rasoi – Restaurant Management System (First Try)


📌 Project Overview

Annapurna Rasoi is a web-based restaurant management system developed using HTML, CSS, and JavaScript. The system provides an interactive platform for customers to explore the menu, add items to the cart, and place orders, while also enabling administrators to manage inventory and monitor orders.

🎯 Objectives
To develop a simple and efficient restaurant management system
To provide a user-friendly interface for food ordering
To implement inventory tracking using frontend technologies
To demonstrate real-world application of web development concepts
✨ Key Features
👤 Customer Features
Browse categorized menu (Starters, Main Course, Drinks, Desserts)
Add items to cart
View total bill
Place orders
🛠️ Admin Features
Add new menu items
Manage inventory (stock updates automatically)
View all placed orders
⚙️ System Features
Real-time stock management
Persistent data using localStorage
Multi-page navigation
Responsive and modern UI
🧰 Technologies Used
HTML – Structure of the website
CSS – Styling and layout design
JavaScript – Functionality and logic
LocalStorage – Data persistence
🚀 How to Run the Project
Download or clone the repository
Open the folder in Visual Studio Code
Install Live Server
Right-click index.html → Open with Live Server
