Annapurna Rasoi - Restaurant Management System

Project Overview
Annapurna Rasoi is a web-based restaurant management system developed using HTML, CSS, and JavaScript. The system provides an interactive platform for customers to browse menu items, add them to a cart, and place orders. It also allows administrators to manage menu items, monitor inventory, and track orders efficiently using localStorage.
Objectives
To develop a simple and efficient restaurant management system
To provide a user-friendly interface for food ordering
To implement inventory tracking using frontend technologies
To demonstrate real-world application of web development concepts
Key Features
Customer Features
Browse categorized menu (Starters, Main Course, Drinks, Desserts)
Add items to cart
View total bill
Place orders
Admin Features
Add new menu items
Manage inventory with automatic stock updates
View all placed orders
System Features
Real-time stock management
Persistent data using localStorage
Multi-page navigation
Clean and responsive user interface
Technologies Used
HTML for structure
CSS for styling
JavaScript for logic and functionality
LocalStorage for data storage
How to Run the Project
